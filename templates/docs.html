API Documentation

POST to /api/predict with JSON payload:

{
    {% for feature in feature_names %}
    "{{ feature }}": {% if not loop.last %},{% endif %}
    {% endfor %}
}

Example: {"worst area": 711.2, "worst concave points": 0.1288, "mean concave points": 0.04781, "worst radius": 15.11, "mean concavity": 0.06664}

Response:

{
    "prediction": "Benign" or "Malignant",
    "confidence": ,
    "features": {
        {% for feature in feature_names %}
        "{{ feature }}": {% if not loop.last %},{% endif %}
        {% endfor %}
    }
}
